전환 효과 (Transitions)

아래로 이동하여 다양한 효과를 확인하세요. (기본: Slide)

Convex

3D 공간에서 볼록한 형태로 회전하는 효과입니다.

Concave

3D 공간에서 오목한 형태로 회전하는 효과입니다.

None

전환 효과 없이 즉시 변경됩니다.

발표자 노트 (Speaker Notes)

키보드에서 'S' 키를 눌러보세요.

발표자만 볼 수 있는 별도의 창에 메모, 다음 슬라이드 미리보기, 타이머 등이 표시되어 발표를 도와줍니다.

이 부분은 발표자만 볼 수 있는 비밀 메모입니다. - 다음 슬라이드는 인포그래픽 예시입니다. - 시각 자료의 중요성을 강조하고, Chart.js 같은 라이브러리를 활용할 수 있다고 언급하세요.

발표자 노트 (Notes)

키보드에서 S 키를 누르면 발표자 노트 창이 열립니다.

이 창은 청중에게는 보이지 않으며, 발표 시간, 다음 슬라이드 미리보기, 발표자 노트 등을 확인할 수 있습니다.

여기는 발표자 노트입니다. 이 슬라이드에서 강조할 점을 적어둘 수 있습니다. - S 키를 눌러 발표자 뷰를 열 수 있음을 시연하기 - 시간 관리의 중요성 언급

테마 및 효과 전환

화면 하단의 컨트롤 패널을 이용해 실시간으로 테마와 전환 효과를 변경할 수 있습니다.

이 기능은 이 데모 페이지에 추가된 커스텀 JavaScript로 구현되었습니다.


// 테마 변경 로직 예시
const themeLink = document.querySelector('#theme-style');
const themeSelector = document.querySelector('#theme-selector');

themeSelector.addEventListener('change', event => {
    const theme = event.target.value;
    themeLink.href = `https://cdn.jsdelivr.net/npm/reveal.js@5.1.0/dist/theme/${theme}.css`;
});

인쇄 및 PDF 내보내기

접근성 및 성능 팁

더 나은 프레젠테이션을 위한 간단한 체크리스트입니다.

  • ✔️ 대비(Contrast): 배경과 텍스트 색상의 대비를 충분히 확보하세요.
  • ✔️ 폰트 크기: 발표 환경에 맞춰 가독성 있는 폰트 크기를 사용하세요.
  • ✔️ 애니메이션: 과도한 애니메이션은 집중을 방해할 수 있으니 꼭 필요할 때만 사용하세요.
  • ✔️ 이미지 최적화: 고용량 이미지는 로딩 속도를 저하시킬 수 있습니다.

접근성 및 성능 팁

  • 접근성: 시맨틱 HTML 태그(<h1>, <p> 등)를 사용하고, 이미지에 alt 속성을 제공하여 스크린 리더 사용자를 배려하세요.
  • 성능: 고화질 배경 이미지나 비디오는 압축하여 사용하세요. loading="lazy" 속성을 이미지에 추가하면 초기 로딩 속도를 개선할 수 있습니다.
  • 가독성: 테마를 선택할 때 배경과 텍스트의 명도 대비가 충분한지 확인하세요.

GitHub Pages로 배포하기

  1. GitHub에 새로운 저장소를 만듭니다.
  2. reveal.js 프로젝트 파일 전체를 push 합니다.
  3. 저장소의 `Settings` > `Pages` 메뉴로 이동합니다.
  4. `Source`를 `Deploy from a branch`로, `Branch`를 `main` (또는 `master`)으로 설정하고 저장합니다.
  5. 잠시 후 공개된 URL로 슬라이드를 확인할 수 있습니다.

Netlify로 배포하기

더 빠르고 간편한 배포 방법입니다.

  1. GitHub에 프로젝트를 push 합니다.
  2. Netlify에 로그인하고 `New site from Git`을 선택합니다.
  3. 배포할 GitHub 저장소를 선택합니다.
  4. 빌드 설정은 비워두고 `Deploy site`를 클릭하면 즉시 배포가 완료됩니다.

인터넷 연결을 확인하거나, 브라우저 콘솔(F12)에서 네트워크 오류를 확인하세요.
플러그인 작동 안 함
Reveal.initialize의 plugins 배열에 플러그인 변수(예: RevealHighlight)가 올바르게 추가되었는지, 스크립트 로드 순서가 맞는지 확인하세요.
코드 하이라이트 안 됨
<pre><code class="language-xxx"> 구조가 맞는지, RevealHighlight 플러그인이 로드되었는지 확인하세요. 언어 이름(js, html 등)이 정확해야 합니다.
PDF 인쇄 시 레이아웃 깨짐
?print-pdf 쿼리 파라미터를 사용했는지 확인하세요. 이 모드는 인쇄에 최적화된 스타일을 적용해 줍니다. 일부 브라우저에서는 페이지 크기 설정이 필요할 수 있습니다.
